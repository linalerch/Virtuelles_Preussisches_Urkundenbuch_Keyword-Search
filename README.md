# Virtuelles_Preußisches_Urkundenbuch_Keyword-Search

Keyword-searcher for the [virtuelles Preußisches Urkundenbuch](https://www.spaetmittelalter.uni-hamburg.de/Urkundenbuch/#). Coded with AI.

In Linux terminal, type:

```bash
python3 -m venv venv

source venv/bin/activate

pip install requests beautifulsoup4

python search.py
```
